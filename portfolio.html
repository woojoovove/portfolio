영화관 관리시스템

Java, Oracle

주    제
영화관 관리 시스템 구현
역    할
요구분석, 관리자 매점 관리 기능 구현, 회원 무비포스트 기능 구현
WHAT

1. 무비포스트 금지어 필터링 기능

  • - 제목과 내용 작성시 금지어가 포함되어 있을 경우 작성이 불가
  • - 사용자에게 필터링 된 금지어 안내

    •     예: "금지어 OOO가 포함되었습니다. 다시 작성해주세요"
WHY
  • - 관리 업무 자동화 및 모니터링 업무 최소화
  • - 이용객의 무비포스트 서비스 만족도 향상
  • - 무비포스트 서비스 활성화를 통한 영업 이익 신장 도모
HOW
  • - 금지어 데이터 파일 입출력: BufferedReader, BufferedWriter
  • - 데이터 조작: ArrayList
  • - 금지어 검색: indexOf(), 사용자 정의 메소드
WHAT

2.무비포스트 작성, 수정, 삭제 기능

  • - 영화 검색 후 포스트 작성 가능
  • - 마이 포스트 열람 가능
WHY
- 마이 포스트 관리
  •    - 무비포스트 서비스 활성화 도모
  •    - 데이터 무결성 보장
- 영화 검색 기능 강화: 포스트 작성시 편의성 증대
- 무비포스트, 무비포스트 평점 → 영화 정보 반영: 구조적인 데이터 관리
HOW
- 무비포스트, 영화 데이터 파일 입출력: BufferedReader, BufferedWriter
- 데이터 조작: ArrayList, 사용자 정의 클래스
WHAT
3.매점 품목별 재고 관리 기능
- 품목(팝콘, 콜라, etc...)별 재고 관리
- 품목 선택 후 상품 등록
WHY
  • - 관리 업무 자동화 및 모니터링 업무 최소화
  • - 이용객의 무비포스트 서비스 만족도 향상
  • - 무비포스트 서비스 활성화를 통한 영업 이익 신장 도모
HOW
- 매점 이용객의 만족도 향상
- 상품 중복 입력 방지 등 재고 관리의 효율성 향상
- 품목별 메뉴판 출력
예:
A001 고소팝콘 B001 콜라 C001 나초
B002 달콤팝콘 B002 제로콜라 B003 사이다
C001 나초 C002 치즈나초 C002 어니언나초

학원 관리 시스템

Java, Oracle

주    제
학원 관리 시스템 구현
역    할
요구분석, ERD 작성, 스키마 구축/관리/활용, 출결 조회 기능 구현, 특별 상담 알림 기능 구현
WHAT
1. 특별 상담 알림 기능
- 출결 점수가 15/20 미만인 교육생: 교사에게 특별 상담 요청 알림 자동 전송
   예: 교사 김OO의 특별상담 대상 학생 조회
이름 담당 교사 출결 점수 상담 여부
박OO 김OO 12.5 N
최OO 김OO 11.5 N
WHY
- 특별 상담을 통해 면학분위기 조성, 교육생의 수강 만족도 향상
- 교육생 출결 관리 업무의 자동화
HOW
- 출결 성적 및 상담 일지 데이터 입출력: JAVA Oracle DB 연동
- 디자인 패턴: MVC
- 데이터 조작: Java: ArrayList, 사용자 정의 클래스
WHAT
2. 출입기록데이터 점수화
   - 출입기록데이터를 분석, 지각/조퇴/출석 부여
   - 시간 단위 점수 세분화
WHY
- 학원 출결관리의 공정성, 정확성 향상
   - 교육생 관리를 통한 면학분위기 조성, 교육생의 수강 만족도 향상
   - 출결관리 업무의 자동화
HOW

- 출입 기록 입출력: JAVA & Oracle DB 연동
- 디자인 패턴: MVC
- 데이터 조작:
   - Java: ArrayList, 사용자 정의 클래스
   - Oracle: GROUP BY, HAVING, OUTER JOIN, CASE
